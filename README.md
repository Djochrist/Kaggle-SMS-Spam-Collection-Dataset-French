# Collection de SMS Spam - Jeu de Données Français

## Contexte

La Collection de SMS Spam constitue un ensemble de messages SMS étiquetés, collectés dans le cadre de recherches sur le spam SMS. Elle comprend 5 453 messages SMS en français, classés comme ham (légitime) ou spam.

Ce jeu de données représente une traduction française du jeu de données original disponible sur Kaggle : https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset. Les jeux de données étant généralement disponibles uniquement en anglais, ce qui n'est pas toujours adapté aux besoins francophones, ce projet vise à fournir une ressource précieuse pour les apprenants en apprentissage automatique souhaitant développer des projets de détection de spam. Les jeux de données en français étant rares, cette initiative comble une lacune importante.

## Contenu

Les fichiers contiennent un message par ligne. Chaque ligne comprend deux colonnes : v1 indique l'étiquette (ham ou spam), et v2 contient le texte brut du message.

## Collaboration

Toute personne souhaitant contribuer en ajoutant de nouveaux messages ou en apportant des ajustements est invitée à forker le projet, le cloner, puis soumettre une pull request.
